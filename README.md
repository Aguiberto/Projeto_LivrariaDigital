# Projeto de gerenciamento de biblioteca

## Objetivos
1. Realizar cadastros:

    - Livros
    - Usuários
    - Empréstimos

2. Controle de devoluções e multas por atrasos.

    - Verificar se o livro já foi emprestados
    - Multa por perda
    - Multa por atraso
    - Multa por dano

3. Gerar relatórios dos livros mais emprestados.
